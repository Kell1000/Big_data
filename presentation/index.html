DS — Pipeline ML

Étapes du Pipeline Data Science

1

Data Extraction — Spark ← HDFS

PySpark charge les fichiers Parquet (posts, comments, post_updates) depuis HDFS. Join posts + updated metrics.

2

EDA — Analyse Exploratoire

Distributions, top subreddits, analyse engagement, corrélations, breakdown media types dans Jupyter.

3

NLP — Sentiment, Émotion & Toxicité

BERT analyse chaque post : sentiment score, émotion dominante, toxicité. Génération des embeddings textuels.

4

Topic Modeling — BERTopic

Découverte automatique des topics avec BERTopic en utilisant les embeddings pré-calculés. Visualisation interactive.

5

ML Training — 4 Modèles Prédictifs

Random Forest, Gradient Boosted Trees & XGBoost. Features : text embeddings + image features + metadata Reddit.
